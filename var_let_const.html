difference between var, let, const

Var

The scope is global when a var variable is declared outside a function. This means that any variable that is declared with var outside a function block is available for use in the whole window.
var is function scoped when it is declared within a function. This means that it is available and can be accessed only within that function.
i.e., function scoped
var can be re-declared within scope and updated
can be declared wihtout initilisation

Let

let is block scoped
let cannot be re-declared within same scope of block but can be updated

but can be re-declared outside the scope
can be declared wihtout initilisation

Const

const are block scoped
const cannot be re-declared or updated within scope
const object variable cannot be updated but the properties of that object can be updated
must be initialised while declared
