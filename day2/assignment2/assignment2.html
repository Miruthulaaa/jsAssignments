For array
1)arr.push(...items) – adds items to the end,
2)arr.pop() – extracts an item from the end,
3)arr.shift() – extracts an item from the beginning,
4)arr.unshift(...items) – adds items to the beginning.
5)delete - remove - makes undefined - array length not altered
6)splice - remove - and length reduced - remove particular index and can remove a number of elements
7)slice - copy from start and end index
For string:
1)fromCharCode(97,98,99,120,121,122)) -- prints unicode caharcter 2)replace('string to be replaced', 'string replaced with') 3)search -- search for instance and returns -1 if not found , returns the start index if found 4)slice(start,end) -copy from start to end 5)subString(start, end) -slice from start to before end
